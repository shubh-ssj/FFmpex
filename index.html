// Core Features

Everything FFmpeg
can do — visually.

A comprehensive dark-themed GUI that surfaces the full power of FFmpeg without requiring you to memorise a single flag. Built with CustomTkinter for a native, snappy feel on Windows, macOS, and Linux.

🎬
Video Conversion
Convert between all major containers — MP4, MKV, AVI, MOV, WebM, and 10+ more — with smart per-container codec defaults.
🎵
Audio Conversion
Transcode to MP3, AAC, FLAC, Opus, WAV, ALAC, and more. Lossless-to-lossy, lossy-to-lossless — full bitrate control from 32k to 512k.
📦
Smart Compression
Target-size presets for Discord, WhatsApp, Telegram, and more. CRF slider ranges from archive-quality (14) to smallest possible file (35).
⚡
Hardware Acceleration
NVIDIA NVENC, AMD AMF, Intel QSV, Apple VideoToolbox, and Linux VAAPI — GPU encoding is automatically detected and offered.
🎯
Platform Presets
One-click export settings for YouTube, TikTok, Instagram, Twitter, Plex, PS5, Xbox, Steam Deck, and 20+ more destinations.
🗂
Job Queue
Queue multiple encoding jobs, run them sequentially with live ETA tracking, and get a desktop notification when the batch finishes.
🖱
Drag & Drop
Drop files directly onto the window to load them instantly. No file picker needed. Requires optional tkinterdnd2.
🎨
Custom Themes
Multiple built-in color themes — Dracula, Nord, and more — built on CustomTkinter's flexible theming engine with full light/dark support.
📐
Resolution Scaling
Scale from 240p all the way up to 8K using smart -2 padding to preserve aspect ratio. Codec quality flags adapt per-encoder automatically.

// Installation

Up and running
in 60 seconds.

FFmpex requires Python 3.9+ and an existing FFmpeg install. All GUI dependencies are on pip — optional extras unlock drag-and-drop, tray icon, and desktop notifications.

Quick Start
# 1. Make sure FFmpeg is installed
# Windows: winget install ffmpeg
# macOS: brew install ffmpeg
# Linux: sudo apt install ffmpeg
 
# 2. Required dependencies
pip install customtkinter pillow
 
# 3. Optional extras
pip install tkinterdnd2 # drag-and-drop
pip install pystray # system tray
pip install plyer # desktop notifications
 
# 4. Run
python ffmpex_v2.py
 
# Override FFmpeg path at launch
python ffmpex_v2.py --ffmpeg /custom/path/ffmpeg
